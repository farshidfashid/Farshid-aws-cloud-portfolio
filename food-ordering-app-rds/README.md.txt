# Full-Stack Food Ordering Web Application with Amazon RDS

## 📌 Project Summary
Designed and deployed a full-stack web application that allows users to browse food items and place orders, with persistent data storage using Amazon RDS.

This project focuses on backend-database integration and structured application architecture in a cloud environment.

## 🏗 Architecture Overview
- Frontend (HTML/CSS/JavaScript)
- Backend API (Node.js / Express)
- Amazon RDS (Relational Database)
- IAM-based access control
- Cloud-hosted deployment

## 🚀 Key Features
- Dynamic food menu retrieval
- Order placement functionality
- Structured relational database schema
- Persistent storage using RDS
- Backend-to-database connectivity

## 🛠 Technologies Used
Node.js  
Express.js  
Amazon RDS  
MySQL / PostgreSQL  
HTML | CSS | JavaScript  

## 📚 Key Learning Outcomes
- Connecting backend applications to relational databases
- Designing normalized database schemas
- Managing secure database credentials
- Handling CRUD operations in production-style environments

## 🔐 Security Considerations
- Database deployed within controlled network access
- Restricted inbound rules for RDS
- Environment variables used for credential management

## 📈 Future Improvements
- Add authentication (JWT or Cognito)
- Implement containerization
- Deploy using ECS or Kubernetes
