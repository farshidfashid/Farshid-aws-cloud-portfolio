# Secure File Storage Web Application

## 📌 Project Summary
Developed a secure file upload and retrieval system leveraging Amazon S3 for storage and API Gateway with Lambda for controlled backend access.

## 🏗 Architecture Overview
- Frontend Web Interface
- Amazon API Gateway
- AWS Lambda (File Handling Logic)
- Amazon S3 (Object Storage)
- IAM Policies for Access Control

## 🚀 Key Features
- Secure file upload endpoint
- Controlled file retrieval
- API-driven architecture
- Serverless backend processing

## 🛠 Technologies Used
Amazon S3  
Amazon API Gateway  
AWS Lambda  
IAM  

## 📚 Key Learning Outcomes
- Secure object storage handling
- API-based file access design
- Access policy enforcement

## 🔐 Security Considerations
- Least privilege IAM roles
- Controlled S3 bucket permissions
- Input validation for uploads

## 📈 Future Improvements
- Add file encryption (SSE-S3 / KMS)
- Implement user authentication
- Add file versioning and lifecycle rules
