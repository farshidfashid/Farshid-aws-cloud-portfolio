# FilmDex – Containerized Full-Stack Movie Review Application

## 📌 Project Summary
Built a multi-container full-stack movie review web application using Node.js and Express for backend services and a dynamic frontend, orchestrated with Docker Compose.

## 🏗 Architecture Overview
- Frontend container (HTML/CSS/JS)
- Backend API container (Node.js + Express)
- Docker networking for inter-service communication
- Docker Compose for orchestration

## 🚀 Key Features
- RESTful API architecture
- Multi-container deployment
- Consistent environment setup
- Single-command startup (docker compose up)
- Isolated service communication via defined ports

## 🛠 Technologies Used
Node.js  
Express.js  
Docker  
Docker Compose  
REST APIs  
HTML | CSS | JavaScript  

## 📚 Key Learning Outcomes
- Containerization fundamentals
- Service orchestration
- Backend/frontend communication
- DevOps workflow principles

## 🔐 Security Considerations
- Isolated container networking
- Defined exposed ports
- Environment variable configuration
