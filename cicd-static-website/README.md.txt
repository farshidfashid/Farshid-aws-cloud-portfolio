# CI/CD Pipeline using GitHub, AWS CodePipeline & CodeBuild

## 📌 Project Summary
Designed and implemented a Continuous Integration and Continuous Deployment (CI/CD) pipeline integrating GitHub with AWS CodePipeline and CodeBuild for automated application deployment.

## 🏗 Architecture Overview
- GitHub (Source Repository)
- AWS CodePipeline (Pipeline Orchestration)
- AWS CodeBuild (Build & Test Stage)
- Deployment Stage (S3 / EC2 / Lambda)

## 🚀 Key Features
- Automated build on code push
- Version-controlled deployments
- Pipeline stage monitoring
- Continuous integration workflow
- Reduced manual deployment effort

## 🛠 Technologies Used
GitHub  
AWS CodePipeline  
AWS CodeBuild  
Amazon S3 / EC2 / Lambda  

## 📚 Key Learning Outcomes
- CI/CD workflow design
- Buildspec configuration
- Automated artifact generation
- Deployment automation strategies

## 🔐 Security Considerations
- IAM roles for pipeline stages
- Secure GitHub integration
- Controlled artifact access

## 📈 Future Improvements
- Add automated testing stage
- Implement blue/green deployment
- Add approval stage for production
