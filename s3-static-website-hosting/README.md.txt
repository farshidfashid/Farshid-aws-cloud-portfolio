# Static Website Hosting using Amazon S3

## 📌 Project Summary
Deployed a static website using Amazon S3 with proper bucket configuration, access control policies, and static hosting features enabled.

## 🏗 Architecture Overview
- S3 bucket configured for static website hosting
- Public access configuration managed via bucket policy
- Custom error and index document setup

## 🚀 Key Features
- Serverless static hosting
- Cost-effective architecture
- Secure bucket policy configuration
- Object-level storage management

## 🛠 Technologies Used
Amazon S3  
IAM Policies  

## 📚 Key Learning Outcomes
- Understanding object storage architecture
- Managing public access securely
- Hosting websites without compute servers

## 🔐 Security Considerations
- Restricted bucket policy to required permissions only
- Public access block configuration reviewed

## 📈 Future Improvements
- Integrate CloudFront CDN
- Add HTTPS support
