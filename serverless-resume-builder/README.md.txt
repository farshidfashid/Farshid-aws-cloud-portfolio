# Serverless Resume Builder with Visitor Counter

## 📌 Project Summary
Developed a fully serverless resume website with a dynamic visitor counter using AWS services. The frontend is hosted on Amazon S3, while visitor tracking is handled via API Gateway, Lambda, and DynamoDB.

This project demonstrates full-stack serverless architecture with persistent backend integration.

## 🏗 Architecture Overview
- Amazon S3 (Frontend Hosting)
- Amazon API Gateway (REST API)
- AWS Lambda (Visitor Count Logic)
- Amazon DynamoDB (Counter Storage)
- CloudWatch Logs (Monitoring)

## 🚀 Key Features
- Static resume website hosting
- Dynamic visitor counter
- REST API integration
- Serverless backend logic
- Scalable NoSQL storage
