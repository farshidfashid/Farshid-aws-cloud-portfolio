# Event-Driven Serverless Automation using AWS Lambda

## 📌 Project Summary
Built an event-driven automation system where AWS Lambda functions are triggered by S3 object uploads and CloudWatch events to perform automated processing tasks.

## 🏗 Architecture Overview
- Amazon S3 (Event Source)
- Amazon CloudWatch (Scheduled/Event Trigger)
- AWS Lambda (Processing Logic)
- CloudWatch Logs (Monitoring)

## 🚀 Key Features
- Automated processing on file upload
- Scheduled background jobs
- Fully serverless architecture
- Logging and monitoring enabled

## 🛠 Technologies Used
Amazon S3  
Amazon CloudWatch  
AWS Lambda  

## 📚 Key Learning Outcomes
- Event-driven architecture design
- Trigger-based automation
- Serverless workflow implementation

## 🔐 Security Considerations
- Restricted Lambda execution role
- Controlled S3 event permissions

## 📈 Future Improvements
- Add Step Functions workflow
- Implement retry mechanisms
- Add error notification system
