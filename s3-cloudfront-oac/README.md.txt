# Secure Content Delivery using CloudFront with Origin Access Control (OAC)

## 📌 Project Summary
Configured Amazon CloudFront with Origin Access Control to securely deliver private S3-hosted content globally via HTTPS.

## 🏗 Architecture Overview
- Amazon S3 (Private Bucket)
- CloudFront Distribution
- Origin Access Control (OAC)
- SSL/TLS Encryption

## 🚀 Key Features
- Global content delivery
- Edge caching optimization
- Secure private origin access
- HTTPS enforcement

## 🛠 Technologies Used
Amazon CloudFront  
Amazon S3  
Origin Access Control  
SSL/TLS  

## 📚 Key Learning Outcomes
- CDN caching behavior
- Securing private origins
- Performance optimization strategies

## 🔐 Security Considerations
- Direct S3 access blocked
- CloudFront-only access enabled via OAC
