# Power of Math – Full-Stack Serverless Web Application

## 📌 Project Summary
Developed a full-stack serverless web application that calculates exponential values using a RESTful backend powered by AWS Lambda and API Gateway, with frontend deployment through AWS Amplify.

## 🏗 Architecture Overview
- AWS Amplify (Frontend Hosting)
- Amazon API Gateway (HTTP API)
- AWS Lambda (Python Backend)
- CloudWatch Logs (Monitoring)
- CORS Configuration

## 🚀 Key Features
- Fully serverless architecture
- REST API integration
- Branch-based CI/CD via Amplify
- Secure CORS handling
- Real-time computation

## 🛠 Technologies Used
AWS Amplify  
Amazon API Gateway  
AWS Lambda (Python)  
CloudWatch  
HTML | CSS | JavaScript  

## 📚 Key Learning Outcomes
- Connecting frontend to serverless backend
- API routing and integration
- Handling cross-origin resource sharing
- Debugging production-style deployments

## 📈 Future Improvements
- Add DynamoDB for calculation history
- Implement Amazon Cognito authentication
