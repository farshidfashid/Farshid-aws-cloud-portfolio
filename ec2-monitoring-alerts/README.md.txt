# Real-Time EC2 Health Monitoring & Alerting System

## 📌 Project Summary
Implemented a real-time monitoring solution that tracks EC2 instance health metrics and sends alerts using Amazon SNS when predefined thresholds are exceeded.

## 🏗 Architecture Overview
- Amazon EC2 (Compute Instances)
- Amazon CloudWatch (Metrics & Alarms)
- Amazon SNS (Notification Service)

## 🚀 Key Features
- CPU utilization monitoring
- Memory usage tracking (via custom metrics)
- Automated alert notifications
- Configurable threshold-based alarms

## 🛠 Technologies Used
Amazon EC2  
Amazon CloudWatch  
Amazon SNS  

## 📚 Key Learning Outcomes
- Implementing observability in cloud systems
- Designing alerting mechanisms
- Monitoring system health metrics

## 🔐 Security Considerations
- Controlled SNS topic permissions
- Secure EC2 access configuration

## 📈 Future Improvements
- Integrate dashboard visualization
- Add auto-scaling trigger policies
- Centralized logging aggregation
