# End-to-End Serverless Feedback Web Application

## 📌 Project Summary
Developed a fully serverless feedback collection system where users submit feedback through a web interface, and the data is stored securely in Amazon DynamoDB.

The system follows an event-driven, scalable architecture with no server management required.

## 🏗 Architecture Overview
- Amazon S3 (Frontend Hosting)
- Amazon API Gateway (REST API)
- AWS Lambda (Backend Logic)
- Amazon DynamoDB (NoSQL Storage)
- CloudWatch Logs (Monitoring)

## 🚀 Key Features
- Fully serverless architecture
- REST API integration
- Real-time feedback storage
- Scalable NoSQL backend
- Logging and monitoring enabled

## 🛠 Technologies Used
Amazon S3  
Amazon API Gateway  
AWS Lambda  
Amazon DynamoDB  
CloudWatch
