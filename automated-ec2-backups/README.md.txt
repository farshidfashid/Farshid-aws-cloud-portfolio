# Automated EC2 Backup & Snapshot Management System

## 📌 Project Summary
Built an automated backup system that creates scheduled EBS snapshots using EventBridge and AWS Lambda to ensure data protection and disaster recovery readiness.

## 🏗 Architecture Overview
- Amazon EC2 (EBS Volumes)
- Amazon EventBridge (Scheduled Trigger)
- AWS Lambda (Snapshot Automation)
- CloudWatch Logs (Monitoring)

## 🚀 Key Features
- Automated snapshot creation
- Scheduled execution via EventBridge
- Logging and monitoring enabled
- Reduced manual operational overhead

## 🛠 Technologies Used
Amazon EC2  
Amazon EventBridge  
AWS Lambda  
CloudWatch  

## 📚 Key Learning Outcomes
- Automating infrastructure operations
- Designing disaster recovery strategies
- Implementing event-based automation

## 🔐 Security Considerations
- Least privilege IAM role for Lambda
- Snapshot access control

## 📈 Future Improvements
- Implement retention policies
- Add cross-region replication
