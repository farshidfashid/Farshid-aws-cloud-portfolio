# Cloud Monitoring & Cost Alerting System

## 📌 Project Summary
Implemented a monitoring and cost alerting system using Amazon CloudWatch, Amazon SNS, and AWS Budgets to track infrastructure health and spending.

## 🏗 Architecture Overview
- CloudWatch Metrics & Alarms
- SNS Notification Topics
- AWS Budgets for Cost Monitoring
- Email alert integration

## 🚀 Key Features
- Real-time performance monitoring
- Threshold-based alerts
- Budget tracking and notifications
- Cost control visibility

## 🛠 Technologies Used
Amazon CloudWatch  
Amazon SNS  
AWS Budgets  

## 📚 Key Learning Outcomes
- Observability best practices
- Cost governance strategies
- Alerting and escalation mechanisms

## 🔐 Security Considerations
- Restricted SNS topic policies
- Controlled alarm configuration

## 📈 Future Improvements
- Centralized monitoring dashboard
- Multi-environment monitoring setup
- Slack integration for alerts
